# 🟢 Fundamentos (UI & UX)

Este repositório reúne projetos focados na construção de interfaces modernas, responsivas e com alta qualidade visual.

## 🚀 Objetivo
Desenvolver domínio em UI/UX, criando experiências visuais que seguem padrões profissionais utilizados por grandes produtos digitais.

## 📦 Projetos incluídos
- Landing Page Premium
- Sistema de Login e Cadastro
- Formulário Inteligente

## 🛠️ Tecnologias
- HTML5
- CSS3 (Flexbox, Grid, Responsividade)
- JavaScript

## 🎯 Foco
- Design moderno e limpo
- Experiência do usuário
- Responsividade
- Microinterações

## 📌 Observação
Todos os projetos foram desenvolvidos com foco em qualidade visual e boas práticas de front-end.

## 📁 Estrutura
Cada projeto está organizado em sua própria pasta com separação de arquivos.

## 💡 Objetivo final
Criar interfaces que transmitam valor e profissionalismo.
