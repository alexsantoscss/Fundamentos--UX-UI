# Cold Creative Auth System

Sistema de autenticação ultra-premium desenvolvido com foco em UI/UX de alto nível e performance para a Cold Creative Studio.

## Tecnologias
- HTML5 Semântico
- CSS3 (Flexbox, Grid, Variables, Glassmorphism)
- JavaScript Vanilla (ES6+)

## Como Executar
1. Clone o repositório.
2. Certifique-se de ter o Node.js instalado.
3. Execute o comando `npm start` para iniciar o servidor de desenvolvimento.

## Padrões de Design
- Tipografia: Poppins & Montserrat
- Paleta: Deep Dark & Purple Neon Gradient
- Efeitos: Backdrop Blur, Transições Easing-in-out
